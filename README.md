# Cafe

A Slack app to help you meet more people across your workspace.

## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
```sh
git clone https://github.com/dijonmusters/cafe.git
```
2. Install NPM packages
```sh
npm install
```
3. Start dev server
```sh
npm start
```
